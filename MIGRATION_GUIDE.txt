## How to Get Started

### Option 1: Quick Start (Recommended)
1. Double-click `START.bat` (Windows)
   - Or: `powershell -ExecutionPolicy Bypass -File START.ps1`
2. A terminal will open and start the server
3. Open browser to `http://127.0.0.1:5000`
4. Login with: `admin` / `admin`


## Troubleshooting

### Issue: "Address already in use"
**Solution:** Another process is using port 5000. Either:
- Close the other application
- Change port in `app.py`: `app.run(port=5001)`

### Issue: "Module not found"
**Solution:** Run `pip install -r requirements.txt` again

### Issue: "Database locked"
**Solution:** Close any other instances and restart

### Issue: CSS/styling looks broken
**Solution:** Hard refresh browser (Ctrl+Shift+Delete) and clear cache
