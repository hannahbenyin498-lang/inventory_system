                            QUICK START
═════════════════════════════════════════════════════════════════════════

STEP 1: Build on Your Computer
────────────────────────────────────────────────────────────────────────
  1. Open command prompt in project folder
  2. Type: build_executable.bat
  3. Press Enter
  4. Wait 2-5 minutes
  5. Check: dist\ChrisEffect.exe exists ✓

STEP 2: Test It Works
────────────────────────────────────────────────────────────────────────
  1. Double-click: dist\ChrisEffect.exe
  2. Login: admin / admin
  3. Verify it works
  4. Close application

STEP 3: Prepare for Distribution
────────────────────────────────────────────────────────────────────────
  Copy to distribution folder:
  ├── ChrisEffect.exe
  ├── install.bat
  ├── README_FIRST.txt
  └── SYSTEM_REQUIREMENTS.txt

STEP 4: Make It Easy to Use
────────────────────────────────────────────────────────────────────────
  Option A: Zip the folder
  Option B: Send ChrisEffect.exe directly
  Option C: Create Windows installer

STEP 5: Share with Users
────────────────────────────────────────────────────────────────────────
  - Email the file
  - Upload to website
  - Put on USB drive
  - Share via file storage

═════════════════════════════════════════════════════════════════════════
                      BUILD TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════

Problem: "Python not found"
  Solution: Install Python 3.9+ from python.org

Problem: "Module not found errors"
  Solution: pip install -r requirements.txt

Problem: "PyInstaller not found"
  Solution: pip install pyinstaller

Problem: "Icon not found"
  Solution: Verify ceicon.ico exists in project folder

Problem: Build fails with permission error
  Solution: Run build_executable.bat as Administrator
            Right-click → Run as Administrator

═════════════════════════════════════════════════════════════════════════
                      SYSTEM REQUIREMENTS
═════════════════════════════════════════════════════════════════════════

For your computer (building):
  • Windows 7, 8, 10, or 11
  • Python 3.9, 3.10, 3.11, or 3.12
  • 1 GB RAM (for building)
  • 500 MB disk space

For user's computers (running):
  • Windows 7, 8, 10, or 11 (32-bit or 64-bit)
  • 512 MB RAM (1 GB recommended)
  • 300 MB disk space
  • NO Python needed
  • NO internet needed


═════════════════════════════════════════════════════════════════════════
                      IMPORTANT NOTES
═════════════════════════════════════════════════════════════════════════

✓ Executable is PORTABLE
  - Can run from USB drive
  - Can run from network share
  - Can be moved to any Windows computer
  - No installation required

✓ Database is LOCAL
  - Each computer has its own store_inventory.db
  - Data does NOT sync between computers
  - Users should backup their database
  - Can export CSV for migration

✓ No Uninstaller Needed
  - Just delete the .exe file
  - No registry changes
  - No system files modified

✓ Windows SmartScreen Warning
  - First run may show security warning
  - Click "More info" → "Run anyway"
  - This is normal for unsigned executables
