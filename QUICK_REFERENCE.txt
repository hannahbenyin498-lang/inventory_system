## ðŸ“¦ Stock Management

### Viewing Inventory
- Click "Stock" tab at bottom
- Table shows: SKU, Name, Qty, Price, Status

### Adding Product
1. Click blue "+" button (bottom right)
2. Enter: SKU, Name, Quantity, Price
3. Status auto-calculated based on quantity

### Searching Products
- Use search bar at top
- Search by product name, SKU, or category
- Results update in real-time

### Editing Product
1. Find product in stock table
2. Click pencil icon
3. Modify details
4. Click save

### Deleting Product (Admin Only)
1. Find product in stock table
2. Click trash icon
3. Confirm deletion

---

## ðŸ’¾ Import/Export

### Export Inventory
1. Go to Stock tab
2. Click "Export" button
3. CSV file downloads automatically
4. File format: `inventory.csv`

### Import Inventory
1. Go to Stock tab
2. Click "Import" button
3. Select CSV file from computer
4. System processes and updates inventory
5. Shows: Imported, Updated, Invalid counts

### CSV Format Required
```
id,sku,name,category,quantity,price,status,image
1,SKU001,Product Name,Category,50,99.99,In Stock,
```

## ðŸ› Troubleshooting

### Product Not Showing?
- Refresh page (F5)
- Check search filters
- Verify quantity > 0

### Can't Login?
- Verify username/password
- Check caps lock
- Try default credentials

### Export not working?
- Check browser settings
- Disable pop-up blockers
- Try different browser

### Slow performance?
- Close other browser tabs
- Refresh page
- Restart server
