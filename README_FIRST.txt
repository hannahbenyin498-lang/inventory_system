Chris Effect Inventory System
==============================

START HERE:
1. Double-click ChrisEffect.exe
2. Login: admin / admin
3. Start managing inventory!


WHAT IS THIS?
=============
An offline inventory management system for Windows.
• Track products & stock levels
• Record sales
• Import/Export data
• User authentication
• No internet required


QUICK START
===========

First Time Use:
1. Run ChrisEffect.exe
2. Database created automatically
3. Login with admin/admin
4. Add your first product!

Load Sample Data:
1. Unzip the package
2. Open command prompt in folder
3. Type: python scripts/load_sample_data.py
4. Adds sample products for testing


SYSTEM REQUIREMENTS
===================
• Windows 7 or later
• 512 MB RAM (1 GB recommended)
• 150 MB disk space
• No internet needed


FEATURES
========
✓ Add/edit/delete products
✓ Track inventory quantities
✓ Record sales transactions
✓ Import/export CSV files
✓ Low stock alerts
✓ User roles (admin/user)
✓ Product images
✓ Sales history


GETTING HELP

FIRST LOGIN
===========
Username: admin
Password: admin

(You can create additional users after login)


BACKUP YOUR DATA
================
Your data is stored in: store_inventory.db
Back this file up regularly!

To backup:
1. Close ChrisEffect
2. Copy store_inventory.db to safe location
3. Done!


SUPPORT
=======
For bugs/issues:
1. Check SYSTEM_REQUIREMENTS.txt
2. Delete store_inventory.db and restart (fresh start)
3. Check README.md for features
